## 🤝 介绍
本插件通过操控bot与浏览器交互（搜索、点击、滑动、滚动、缩放、输入、切换标签页、收藏等等），  
运行时，bot在后台打开一个浏览器，每完成一个交互动作，bot返回一张浏览器界面的截图。


## 📦 安装

### 第一步，安装本插件
直接在astrbot的插件市场搜索astrbot_plugin_browser，点击安装，等待完成即可
